## Project Information 
.MainApp — The entry point of the app. It runs the program by collecting user input (name, username, password, phone number), passing it to Login for validation, and then handling registration and login.
.Login — Handles all the logic. It validates the username, password, and phone number, registers the user by storing their details, and verifies login credentials. It also returns feedback messages for each action.
.LoginTests — Unit tests for the Login class. It tests that username, password, and phone number validation correctly accept valid inputs and reject invalid ones, and that login succeeds or fails as expected.
---
## Features 
1. Input Validation
Checks that the username, password, and phone number meet the required format before accepting them.
2. While Loops
Keeps asking the user to re-enter their input until it is valid.
3. Unit Testing (JUnit 5)
Automatically tests the validation methods to confirm they correctly accept valid inputs and reject invalid ones.
